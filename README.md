# Tascade

A modern, minimalist task management web application built with **HTML, CSS, and JavaScript**. Tascade helps users organize, prioritize, and track their daily tasks through a clean interface with persistent local storage.

---

## ✨ Features

- Create new tasks
- Edit existing tasks
- Mark tasks as completed
- Delete tasks
- Task categories
- Priority levels
- Due dates
- Local Storage persistence
- Live task statistics
- Responsive design
- Toast notifications
- Modern dark UI

---

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript
- Font Awesome

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/xEhtisham/Syntecxhub_tascade.git
```

Navigate to the project:

```bash
cd tascade
```

Open `index.html` in your browser, or use a local development server such as VS Code Live Server.

---

## 💾 Data Storage

All tasks are stored locally using the browser's Local Storage. No external database or backend is required.
